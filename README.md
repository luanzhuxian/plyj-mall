# plyj-mall

## 安装依赖
```
npm install
```

### 用于本地启动
```
npm run start
```
### 用户开发环境的编译
```
npm run build:dev
```

### 用户生产环境的编译
```
npm run build
```
### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### 开发时登录
```
修改 vue.config.js 文件中的 TOKEN 变量
```
